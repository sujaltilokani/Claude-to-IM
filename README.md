## 📋 What is Claude-to-IM?

Claude-to-IM acts as a bridge between the Claude Code SDK and popular messaging platforms. It lets you integrate AI functions into apps like Telegram, Discord, and Feishu without any coding. 

The app is host-agnostic, which means it does not depend on any specific server or cloud provider. You can run it on your own Windows PC easily.

### Key Features

- Supports Telegram, Discord, and Feishu messaging apps
- Works on Windows PCs without complex setup
- Connects Claude Code SDK to messaging platforms
- Uses TypeScript for reliability and maintainability
- Lightweight and fast with low resource needs

## 🛠 How to Use Claude-to-IM

This section explains the basic steps to link Claude-to-IM with messaging platforms.

### Connect to Your Messaging App

1. Open Claude-to-IM.
2. Choose the messaging service you want to connect: Telegram, Discord, or Feishu.
3. Follow the on-screen prompts to log in to your messaging account. This usually involves entering an API key or logging in via OAuth.
4. Once connected, the app will start forwarding messages between Claude Code and your chat app.

---

### Send Commands to Claude Code

You can now use your chosen messaging app to send commands or ask questions. Claude-to-IM will forward these to Claude Code and send back the answers.

## 🛑 Troubleshooting Common Issues

- **App won't start:** Check if your antivirus or firewall is blocking the app. Try running as administrator.
- **Can't connect to messaging app:** Verify your API key and internet connection.
- **Messages not forwarding:** Restart Claude-to-IM and your messaging app.
- **Installation fails:** Make sure you have enough disk space and your Windows is up to date.

If problems persist, check the README or Issues tab on the GitHub repository.
